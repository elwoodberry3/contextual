# Flow Editor

The Flow Editor is the visual workspace where you build and test flows in Contextual. It provides a drag-and-drop palette of nodes, a canvas for wiring logic, and a right sidebar for tools like SolutionAI.

## Key Areas

* **Left palette** - Browse node categories and drag nodes onto the canvas.
* **Canvas** - Arrange nodes, connect wires, and edit logic.
* **Right sidebar** - Access SolutionAI and other panels.

For help using the assistant panel, see [SolutionAI in the Flow Editor](/documentation-and-resources/components-and-data/flows/node-red-flow-editor/solution-ai.md).
