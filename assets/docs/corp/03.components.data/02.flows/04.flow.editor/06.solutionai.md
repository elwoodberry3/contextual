# SolutionAI

SolutionAI is an assistant panel that helps you build and refine flows directly inside the Flow Editor. It can suggest nodes, explain configuration, and answer questions while you work.

## Where to Find It

Open any Flow and click **Edit Flow** to launch the Flow Editor. In the right sidebar, select the **SolutionAI** tab to open the panel.

## What You Can Do

* Ask questions about nodes, Flow structure, and configuration.
* Get suggested nodes or changes and insert them into your flow.
* Start a new thread at any time using **New Conversation**.

## Web Search Support

SolutionAI can perform web search to pull in fresh sources when answering questions in the Flow Editor, Design, and Data Model experiences.

## Workspaces Per Flow

Each flow has its own SolutionAI workspace. When you switch flows, the conversation history stays scoped to the flow you are editing.
