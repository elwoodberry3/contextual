# Features

Depending on your [service plan](https://www.contextual.io/pricing), you have two optional features you can enable for your Object Types.

#### Audit Trail

Audit Trails provide a comprehensive history of all changes to a given record.

#### Record Versioning

Record versioning allows for tracking of historic versions of records and roll-back to previous data.
