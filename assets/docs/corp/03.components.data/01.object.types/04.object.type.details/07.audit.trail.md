# Audit Trail

You can enable an Audit Trail on your Object Types that captures every operation on a given record instance. The Audit Trail is enabled in the Features tab of the Object Type definition.

<figure><img src="/files/NLoqZCyBMJ3iB910buIj" alt=""><figcaption><p>Object Type Features</p></figcaption></figure>

Once enabled, the Audit Trail and history of changes to records will appear under the Audit Trail tab.
