# Client Grant

## Client Grant Connection

### **Description**

The "Client Grant" connection type is used for OAuth 2.0 Client Credentials Grant authentication. In this method, a Client ID and Client Secret are exchanged for an authentication token. The authentication token is then used as a Bearer token. Tokens can have varying expiration times (sometimes as short as an hour) and must be renewed periodically using the Client ID and Secret.

### **Required Information**

* **API Endpoint:** The URL of the API endpoint.
* **Client ID**: The ID for the client application.
* **Client Secret**: The secret key for the client application.
* **Token Endpoint**: The URL to obtain the token.
