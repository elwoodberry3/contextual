# Types of Agents

In Contextual, agents are designed to handle different kinds of triggers and execution patterns. The primary flow-based agent types are:

* [Event to Flow](/documentation-and-resources/components-and-data/agents/types-of-agents/event-to-flow.md) - process asynchronous events sent by triggers, actions, or other agents
* [Cron to Flow](/documentation-and-resources/components-and-data/agents/types-of-agents/cron-to-flow.md) - execute a flow on a schedule
* [HTTP to Flow](/documentation-and-resources/components-and-data/agents/types-of-agents/http-to-flow.md) - handle inbound HTTP requests

Each type is optimized for a different kind of ingress. Choosing the right agent type helps keep your solution predictable, scalable, and easier to operate.
