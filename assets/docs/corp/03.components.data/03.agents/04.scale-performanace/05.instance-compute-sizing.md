# Instance Compute Sizing

Agent instances can be one of 5 sizes, from Small to XXL. Each agent size comes with unique CPU (mCPUs) and memory (MB/GB). Each Agent size has distinct per hour prices based on your [service tier](https://www.contextual.io/pricing).
